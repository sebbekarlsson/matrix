# matrix
> Tiny matrix library for C

## Usage
> How to use:

    #include <matrix/matrix.h>

    matrix4x4 mat = init_matrix4x4();
    
    translate_matrix4x4(
        mat,
        3.5f,    // x
        122.0f,  // y
        0.0f     // z
    );


## Install
> To install:

    make
    sudo make install
